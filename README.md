# 🌅 Fade In Logo — Flutter

A Flutter demo comparing **explicit** and **implicit** animation approaches using a Flutter logo fade effect.

---

## ✨ Features

- **Explicit Animation** — Uses `AnimationController` + `FadeTransition` with manual forward/reverse control
- **Implicit Animation** — Uses `AnimatedOpacity` + `AnimatedContainer` driven by a boolean state toggle
- Both animations use a 2-second `easeIn` curve

---

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/flutter-fadein-logo.git
cd flutter-fadein-logo
flutter pub get
flutter run
```

---

## 🏗️ Structure

```
lib/
└── main.dart
    ├── HomePage         # Splits screen between both demos
    ├── ExplicitFadeIn   # AnimationController + FadeTransition
    └── ImplicitFadeIn   # AnimatedOpacity + AnimatedContainer
```
